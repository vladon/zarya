# Zarya 0.30.0 Beta

This is a feedback-driven beta update after the public beta preparation build.

## Recommended mode

Use Xray system-proxy mode.

## Downloads

- Windows x64 portable ZIP
- macOS arm64 ZIP
- macOS x64 ZIP if available
- Linux x64 tar.gz

## Verify downloads

Use SHA256 checksums from `SHA256SUMS.txt`.

## Changes since 0.29

- improved diagnostics (Core Manager, subscriptions, first-run, packaging)
- **Help → Copy Support Summary** for safer issue reports
- public beta triage docs and label definitions
- stronger experimental TUN and kill switch warnings
- startup recovery links to diagnostics and reporting guide
- hardened artifact verification

## Experimental

TUN, zarya-helper, and kill switch remain experimental.

## Reporting bugs

Use **Help → Create Diagnostics Bundle**.

Do not post raw proxy links or subscription URLs.
