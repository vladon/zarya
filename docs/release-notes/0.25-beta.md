# Zarya 0.25.0 Beta — Release packaging

## Summary

Release packaging cleanup for reproducible beta artifacts on Windows, macOS, and Linux.

## Added

- Central `BuildInfo` (version, commit, build date, channel, Qt, compiler)
- `Zarya --version` and `zarya-helper --version`
- Packaging scripts with `release-manifest.json` and SHA256 checksums
- Unified portable layout (`translations/`, `docs/`, `cores/`, `data/`, `runtime/`)
- `scripts/smoke-package.py` and platform smoke wrappers
- `docs/release-packaging.md`, `THIRD_PARTY_NOTICES.md`, `RELEASE_NOTES.md`

## Not included

- Code signing and macOS notarization
- Installers, winget/homebrew/apt publishing
- Bundled Xray/sing-box binaries
