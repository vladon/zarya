# Zarya 0.23 Beta

## What works

- Xray system-proxy mode
- Profile/subscription import
- VMess/VLESS/Trojan/Shadowsocks runtime
- Routing/DNS profiles
- Geo data manager
- Core manager
- Backup/import
- Diagnostics bundle

## Experimental

- sing-box TUN mode
- zarya-helper
- Linux nftables kill switch
- Windows WFP kill switch

## Known limitations

- macOS kill switch unsupported
- TUN mode is not production-ready
- No app self-update
- No signed installers yet
- No full localization yet

## How to report issues

- Create Diagnostics Bundle
- Include steps to reproduce
- Do not send raw profile links

## UX (0.23)

- First-run setup wizard
- Status dashboard and empty states
- Improved Start/Stop/Exit flows
- Log panel filters
- Beta warning banner
