# Zarya 0.27.0 Beta

## Focus

Signed builds investigation and signing-ready packaging hooks.

## Added

- Optional Windows Authenticode signing script (`scripts/sign-windows.ps1`)
- Optional macOS signing/notarization script (`scripts/sign-macos.sh`)
- Optional Linux GPG/minisign hooks (`scripts/sign-linux.sh`)
- Release artifact verification script (`scripts/verify-release-artifacts.py`)
- Signing documentation under `docs/signing/`
- Release manifest `signing` fields
- About dialog build integrity section (signed/unsigned)

## Still unsigned by default

0.27 beta artifacts may still be unsigned.
Use SHA256 checksums to verify downloads.

## No app self-update yet

Auto-update and production certificate management remain out of scope.

## Packaging

```powershell
.\scripts\package-windows.ps1 -Configuration Release -OutputDir .\dist -SkipSigning
```

Optional signing (requires credentials):

```powershell
.\scripts\package-windows.ps1 -Sign -SigningIdentity <thumbprint>
```
