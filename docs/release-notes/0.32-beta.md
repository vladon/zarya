# Zarya 0.32.0-beta

## Focus

App self-update design.

## Added

- App update manifest parser and channel policy
- **Help → Check for App Updates…** dialog
- Download-and-verify flow (SHA-256, no install)
- Settings → App updates (separate from Core updates)
- `scripts/generate-update-manifest.py`
- Updater design docs in `docs/updater/`
- `InstallationMode` and app updater status in diagnostics

## Not implemented

- Automatic app replacement
- Silent background updates
- Helper service update
- Signed update enforcement
- Mandatory updates

## Manual test

```bash
python scripts/generate-update-manifest.py \
  --version 0.33.0-beta \
  --channel beta \
  --dist-dir dist \
  --base-url file:///absolute/path/to/dist \
  --output update-manifest.json
```

In Zarya: **Help → Check for App Updates…** → **Choose Local Manifest…** → **Check Now** → **Download and Verify**.
