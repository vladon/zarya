# Zarya 0.24 — Localization baseline

## Summary

Introduces Qt translation infrastructure with English as the source language and Russian as the first target locale.

## Features

- `LanguageManager` with system / English / Russian selection in Settings → General
- Compiled `.qm` catalogs shipped beside the executable (`translations/`)
- `TranslatableEnums` for centralized enum/status labels
- Qt plural forms for count strings
- `scripts/check-translations.py` for CI and local validation
- [docs/localization.md](../localization.md) for developers and translators

## Notes

- Full UI retranslation after language change requires restart
- Protocol names, error codes, and raw core output remain untranslated
- Russian copy is practical baseline quality; not a professional editorial pass
