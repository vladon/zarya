# Zarya 0.31.0-beta

## Focus

Production installer planning.

## Added

- Installer strategy docs (`docs/installer/`)
- Windows WiX skeleton (`packaging/windows/wix/`)
- macOS DMG/PKG strategy docs and script placeholders
- Linux AppImage/deb/rpm skeletons
- Installed layout specification
- Portable-to-installed migration design and **File → Import from Portable Zarya Folder…**
- Helper service installation design
- Installer security model
- `InstallationMode` detection in About, diagnostics, and support summary
- Packaging metadata placeholders

## No production installer yet

0.31 does **not** replace portable beta artifacts.

## Recommended path

Use Xray system-proxy mode.
