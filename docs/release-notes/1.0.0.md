# Zarya 1.0.0

Zarya 1.0.0 is the first stable release focused on the Xray system-proxy desktop client path.

## Stable features

- Xray system-proxy runtime
- Profile management
- Subscription management
- VLESS, VMess, Trojan, and Shadowsocks via Xray
- Routing profiles
- DNS profiles
- Xray Geo Data Manager
- Core Manager for Xray and sing-box
- Node testing
- Tray and safe shutdown
- Startup recovery
- Backup and import
- Diagnostics bundle
- English and Russian UI
- Portable release artifacts

## Recommended mode

Use Xray system-proxy mode.

## Experimental features

The following features exist but are not part of stable support and are disabled/hidden by default:

- sing-box TUN mode
- zarya-helper privileged mode
- Linux nftables kill switch PoC
- Windows WFP kill switch PoC
- helper service installation
- Windows MSI installer PoC
- app self-update install PoC

## Known limitations

- System proxy mode is not VPN mode.
- Some applications may ignore OS proxy settings.
- macOS kill switch is not implemented.
- Builds may be unsigned depending on release process.
- App self-update install is not enabled for stable builds.
- Production installer/service flow is not part of 1.0.0.

## Reporting issues

Use **Help → Create Diagnostics Bundle**.

Do not post raw proxy links, subscription URLs, passwords, helper tokens, or unredacted generated configs.
