# Zarya 0.26.0 Beta — Bugfix pass

## Summary

First beta-quality candidate. Focus on regression fixes, recovery, packaging validation, and documentation — no new features.

## Fixes

- **Choose existing binary** opens a file dialog (wizard, error actions)
- **System proxy recovery** persists pre-enable state to disk; startup recovery can restore or clear Zarya-owned proxy
- **Helper path policy** allows configured sing-box executable parent directory
- **First-run wizard** blocks “start now” without Xray/profile; fetches subscription on finish
- **Startup recovery** adds kill switch recovery option
- **Kill switch settings** “How it works” instead of misleading “Enable Now”
- **TUN warnings** no longer claim kill switch is unimplemented

## Added

- `docs/beta-bug-triage.md`, `docs/beta-regression-checklist.md`, `docs/platform-test-matrix.md`, `docs/beta-todo-audit.md`
- `scripts/run-smoke-tests.py` — C++ smoke + packaging orchestration

## Beta status

Recommended path: **Xray system proxy**. TUN/helper/kill switch remain experimental.
