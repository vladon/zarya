# Zarya 0.35.0 Beta

## Focus

App updater implementation PoC for **portable mode only**.

## Added

- `zarya-updater` executable
- Portable update plan format (`runtime/app-updates/pending-update.json`)
- Windows portable update PoC: download → verify → stage → external updater → replace → post-check → rollback
- Linux portable update (best-effort; AppImage remains download-and-verify only)
- Staging and rollback flow
- Update success/failure startup notices (`--post-update`, `--update-rollback`)
- Updater diagnostics in support bundle
- Package inclusion of `zarya-updater`

## Not supported

- installed/MSI automatic update
- helper service update
- macOS `.app` self-replacement
- AppImage self-replacement
- silent updates
- mandatory updates
- delta updates
- production auto-update

## Manual verification

See [docs/updater/portable-update-implementation.md](../updater/portable-update-implementation.md) and [docs/updater/recovery.md](../updater/recovery.md).

Portable ZIP remains the recommended beta distribution. MSI installed mode can check/download updates but cannot auto-install yet.
