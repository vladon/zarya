# Zarya 0.36.0-rc1

## Focus

First release candidate hardening.

## Stable candidate path

- Xray system-proxy mode
- Profile/subscription management
- Xray routing/DNS profiles
- Core Manager
- Backup/import
- Diagnostics bundle
- Tray/safe shutdown
- English/Russian UI
- Portable artifacts

## Gated experimental features

The following are disabled by default in RC builds:

- sing-box TUN
- zarya-helper privileged mode
- Linux nftables kill switch
- Windows WFP kill switch
- app self-update install PoC

## Known limitations

- system proxy is not VPN mode
- some apps ignore OS proxy settings
- builds may be unsigned
- Windows MSI remains a PoC
- macOS kill switch is not implemented

## Upgrade from 0.35 beta

- Existing settings are preserved.
- If TUN was configured, RC falls back to Xray system-proxy until experimental features are re-enabled.
- App update check and download/verify remain available; install is disabled by default.
