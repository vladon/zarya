# Zarya 0.28.0 Beta

## Focus

Installer/service installation design for zarya-helper.

## Added

- Helper service management abstraction
- Windows `ZaryaHelper` service PoC (install/start/stop/uninstall)
- Minimal Windows service entrypoint in `zarya-helper --service`
- Linux systemd/polkit skeleton files
- macOS SMAppService design documentation
- Helper service status in Settings and diagnostics (`helper-service-status.json`)
- Service recovery documentation
- Helper CLI: `--service-status`, `--service-self-test`, `--service-install-plan`

## Still experimental

- TUN mode
- helper service (optional; not required for system-proxy mode)
- kill switch

## Not included

- production MSI/pkg/deb/rpm installers
- mandatory helper installation
- macOS notarized helper deployment
