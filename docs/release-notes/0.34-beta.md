# Zarya 0.34.0-beta

## Focus

Windows MSI installer proof of concept.

## Added

- WiX-based MSI PoC (`scripts/package-windows-msi.ps1`)
- Installed-mode layout under Program Files
- Start Menu shortcut; optional Desktop shortcut
- Optional helper service install (`INSTALLHELPERSERVICE=1`)
- Windows registry install markers + installed-mode detection
- MSI smoke test script
- [docs/installer/windows-msi-poc.md](../installer/windows-msi-poc.md)
- First-run prompt to import from portable folder (installed mode)

## Still recommended

**Portable ZIP** remains the recommended beta distribution.

## Not included

- Production-signed installer
- Wintun driver installer
- Automatic app self-update
- macOS/Linux installers
- Silent enterprise deployment
