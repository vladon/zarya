# Zarya 0.36 RC Scope

## Included in RC stable path

- Xray system-proxy runtime
- Profile management
- Subscription management
- VLESS / VMess / Trojan / Shadowsocks via Xray
- Routing profiles for Xray
- DNS profiles for Xray
- Xray Geo Data Manager
- Core Manager for Xray/sing-box
- Node testing
- Tray and safe shutdown
- Startup recovery
- Backup/import
- Diagnostics bundle
- English/Russian UI
- Portable packages
- Windows MSI installer PoC as non-default

## Excluded from stable support

- sing-box TUN mode
- zarya-helper production use
- Linux nftables kill switch
- Windows WFP kill switch
- macOS kill switch
- production installer/service installation
- production app self-update
