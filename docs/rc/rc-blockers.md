# RC Blockers

## Blocking

- app cannot start on primary platform
- Xray cannot be installed/detected
- valid Xray profile cannot start
- system proxy cannot be restored
- diagnostics leak secrets
- backup/import data loss
- portable artifact contains user data or runtime secrets
- first-run wizard blocks setup
- crash on safe shutdown

## Non-blocking if documented/gated

- TUN bugs
- kill switch bugs
- helper service bugs
- MSI PoC bugs
- app updater install PoC bugs
