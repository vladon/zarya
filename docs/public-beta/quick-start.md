# Quick Start

## Windows

1. Download `Zarya-0.29.0-beta-windows-x64-portable.zip`.
2. Extract it to a folder you control.
3. Run `Zarya.exe`.
4. Complete the setup wizard.
5. Install Xray through **Tools → Core Manager**.
6. Paste a profile link or add a subscription.
7. Click **Start**.

## macOS

This beta build may be unsigned and not notarized.
macOS may show a warning.

1. Download the macOS archive.
2. Extract `Zarya.app`.
3. Start the app.
4. Install Xray through **Core Manager**.
5. Add a profile or subscription.
6. Click **Start**.

## Linux

1. Download the Linux tarball.
2. Extract it.
3. Run `./zarya`.
4. Install Xray through **Core Manager**.
5. Add a profile or subscription.
6. Click **Start**.

## Recommended settings

- **Runtime:** Xray system proxy
- **Routing:** Bypass LAN
- **DNS:** System DNS
- **TUN:** disabled
- **Kill switch:** disabled
