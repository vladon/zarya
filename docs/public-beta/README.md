# Zarya Public Beta

Zarya is a cross-platform desktop proxy client for Windows, macOS, and Linux.

## Recommended beta path

Use **Xray system-proxy mode** first.

Experimental TUN mode, zarya-helper, and kill switch are available for testing, but they are not production-ready.

## Quick start

1. Download the artifact for your platform.
2. Verify the SHA256 checksum.
3. Start Zarya.
4. Open **Core Manager** and install Xray.
5. Import a profile link or add a subscription.
6. Use **Routing: Bypass LAN**.
7. Use **DNS: System DNS**.
8. Click **Start**.

## If something breaks

Use **Help → Create Diagnostics Bundle** and attach it to your issue.

Do not paste raw proxy links, passwords, or subscription URLs into public issues.
