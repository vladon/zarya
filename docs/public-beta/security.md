# Security

## Reporting security issues

Do not report sensitive security issues in public issues.
Use the private contact channel defined by the project maintainers.

## Sensitive data

Never post raw proxy links, subscription URLs, credentials, helper tokens, or unredacted configs publicly.

## Helper

zarya-helper is experimental.
Install or run it only from a trusted Zarya build.

## Unsigned builds

Beta builds may be unsigned.
Verify SHA256 checksums before running.
